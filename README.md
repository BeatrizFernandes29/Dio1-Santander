# Dio1-Santander

No primeiro desafio pensei em criar desenho de fluxo de diagrama de armazenamento com AWS usando ferramentas S3 e Lambda, passando pelas etapas de upload, processamento e armazenamento.
Neste caso, um usuário envia um arquivo para o Amazon S3. Isso aciona uma função AWS Lambda. A Lambda processa o arquivo, validando-o e salva o resultado em outro bucket S3.
